Privacy Policy for YouExtend YouExtend is a Chrome extension that helps users hide unwanted videos on the YouTube home feed using user-defined topics and keywords. Data we process YouExtend processes visible content on YouTube pages, such as video titles and channel names, in order to determine whether a video matches the user’s filters. YouExtend also stores user-created settings, such as topics, keywords, and filter states, using Chrome extension storage. How data is used The processed page content is used only to provide the extension’s filtering functionality on YouTube pages. User settings are used only to remember the user’s preferences and apply filters across sessions. Data sharing YouExtend does not sell user data. YouExtend does not transfer user data to third parties. YouExtend does not use user data for advertising, profiling, or unrelated purposes.
