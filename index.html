Privacy Policy for YouExtend

YouExtend is a Chrome extension that helps users hide unwanted videos on the YouTube home feed using user-defined topics and keywords.

What YouExtend does

YouExtend reads visible content on YouTube pages, such as video titles and channel names, in order to determine whether a video matches the user’s filters.

Data stored

YouExtend stores user-created settings, such as topics, keywords, and filter states, using Chrome extension storage on the user’s browser.

How data is used

The extension uses visible YouTube page content only to apply the user’s filters and hide matching videos. Stored settings are used only to remember the user’s preferences across browser sessions.

Data sharing

YouExtend does not sell user data. YouExtend does not transfer user data to third parties. YouExtend does not use user data for advertising or any purpose unrelated to the extension’s filtering functionality.

Remote code

YouExtend does not use remotely hosted executable code.
